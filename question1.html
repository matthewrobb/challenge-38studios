Question 1-a: Create an anchor using HTML and then in JavaScript.


HTML:


<a></a>

JavaScript:


var anchor = document.createElement("A");

Question 1-b: Have this anchor reference www.38studios.com.


HTML:


<a href="http://www.38studios.com">38 Studios</a>

JavaScript:


var anchor = document.createElement("A");

anchor.setAttribute("href", "http://www.38studios.com");
anchor.innerHTML = "38 Studios";

Question 1-c: What would happen if the anchor reference was "#38studios" and then clicked on?


The browser would look for an element in the page that has an ID attribute equal to everything to the right of the hash symbol. If it finds one it will scroll the page to that element as well as appending "#38studios" to the end of the url.
